# Requirements Definition

## 1. システム概要
- Webシステムである
- 株価に影響がある情報を整理し、投稿文として自動配信する
- トレーダーが知りたい株価アノマリー情報を整理し、投稿文として自動配信する

## 2. 業務要件
- 株価に影響があるニュース・イベント・指標情報を収集し、投稿候補として整理できること
- トレーダー向けに株価アノマリー情報を整理し、投稿候補として生成できること

## 3. 機能要件
### 3.1 投稿管理
- 投稿を自動で予約作成できること
- 投稿内容を手動で変更できること
- 即時投稿できること
- 投稿予約をキャンセルできること

### 3.2 一覧表示
- 一覧画面で投稿予定日時を確認できること
- 投稿状態（予約済み、投稿済み、キャンセル）を確認できること

## 4. 画面要件
- 投稿一覧画面
- 投稿編集画面
- 投稿確認画面

## 5. 非機能要件
- Webブラウザで利用できること
- 投稿予定時刻に自動実行されること
- 操作履歴と投稿履歴を追跡できること

## 6. 未確定事項
- 投稿先プラットフォーム（X 固定か、他SNS含むか）
- 株価アノマリーの定義ルール
- ニュース/イベントの収集元
- 投稿承認フローの有無
